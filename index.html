Alex Hunt - Game Programmer

Hello! I'm a senior student at Worcester Polytechnic Institute studying Computer Science and Interactive Media and Game Development, planning to graduate in December 2021. I have a passion for UX design and implementation, gameplay programming, and game audio programming.

Crossroads

A 2D top-down adventure game in which you try to escape the circles of hell! Created with Unity Engine.

Nova

A 3D rhythm shooting game featuring built-in latency compensation. Created in Unreal Engine 4 and deployed to Google Stadia's cloud gaming platform to test the effectiveness of latency compensation in cloud gaming platforms.

Cartograbird

A 3D adventure game developed in Unreal Engine 4. Play as Carmen the Cartographer and map a large island in the sky, flying around and collecting items as you go!

Night Run

A 2D game developed in Perlenspiel. Use your flashlight to navigate the dark forest and collect beads of wisdom to find your way to enlightment.

Alex's Audiovisualizer

A customizable web audiovisualizer using HTML Canvas, Web Audio API, and dat.gui.

Web Video Player
